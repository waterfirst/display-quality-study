# Display Quality Study Guide 2026

> 디스플레이 제조업 품질 엔지니어를 위한 종합 학습 가이드
> 산업 횡단 이직까지 커버하는 품질 커리어 로드맵

## 대상
- OLED 개발 → 품질팀 전환 엔지니어
- TFT 소자 품질 담당자
- Automotive 디스플레이 품질 관리자
- 타 산업(반도체, 2차전지, 의료기기 등)으로 이직 준비 중인 품질 엔지니어

## 학습 로드맵
1. **기초 (1~2주)** — TFT 물리, LTPS/IGZO/LTPO, OLED 구조
2. **측정/분석 (2~4주)** — I-V, VT, NBTI/PBTI, Mueller Matrix
3. **불량/신뢰성 (4~8주)** — Mura, Pixel, 신뢰성 가속시험, Automotive
4. **품질 시스템 (8~12주)** — SPC, DMAIC, FMEA, MSA, DOE, 8D, Gate Review, 수율 관리, 품질 비용
5. **커리어 확장 (12주~)** — ASQ CQE, 6시그마 벨트, 타 산업 QMS, Python/SQL, AI/ML

## 주요 콘텐츠 (10개 탭)

- 📋 학습 로드맵 — 5단계 Phase별 학습 경로
- 🔬 TFT 소자 기초 — MOSFET→TFT, LTPO, 핵심 파라미터
- ⚡ 측정/분석법 — I-V, VT 추출, 스트레스 테스트, Mueller Matrix
- 🔴 불량 모드 — Pixel, Mura, Line, 신뢰성 불량, 검사 방법
- 📊 품질 도구 — SPC, DMAIC, FMEA, 8D, MSA, DOE, CoQ, Gate Review
- 🏭 수율 관리 — Yield Waterfall, Loss 분류, OEE, RTY
- 🚗 Automotive 품질 — AEC-Q, IATF 16949, 신뢰성 시험
- 🔄 산업 횡단 커리어 — 이직 경로, 인증 비교, 자격증, 면접 대비
- 📚 참고 자료 — 교과서, 온라인 강의, 학회, 표준, 논문 DB
- ✅ 학습 체크리스트 — 진행률 추적 (localStorage 저장)
