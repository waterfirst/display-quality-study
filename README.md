# Display Quality Study Guide 2026

> 디스플레이 제조업 품질 엔지니어를 위한 종합 학습 가이드

## 대상
- OLED 개발 → 품질팀 전환 엔지니어
- TFT 소자 품질 담당자
- Automotive 디스플레이 품질 관리자

## 학습 로드맵
1. **기초 (1~2주)** — TFT 물리, LTPS/IGZO/LTPO, OLED 구조
2. **측정/분석 (2~4주)** — I-V, VT, NBTI/PBTI, Mueller Matrix
3. **불량/신뢰성 (4~8주)** — Mura, Pixel, 신뢰성 가속시험, Automotive
4. **품질 시스템 (8~12주)** — SPC, DMAIC, FMEA, 8D, Gate Review
